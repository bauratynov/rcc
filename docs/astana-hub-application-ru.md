# Заявка на Astana Hub Seed Money

## Название проекта
rcc — Компилятор языка C на Rust

## Описание
rcc — компактный (8700 строк) компилятор языка C11, написанный на Rust с нулевыми зависимостями. Включает SSA-оптимизации, 3 бэкенда (x86-64, WebAssembly Text, WASM binary), встроенный LSP-сервер для интеграции с IDE, линтер кода и 11 оптимизаций. Компилирует 100% тестов chibicc.

## Проблема
Существующие учебные компиляторы либо слишком примитивны (не учат настоящей компиляторной инженерии), либо слишком сложны (LLVM — 15 миллионов строк). Нет ни одного компилятора который одновременно: маленький, полнофункциональный, с SSA IR, WASM выводом и LSP сервером.

## Решение
rcc — единственный в мире учебный компилятор C с такой комбинацией фич в 8.7K строк. Может использоваться для:
- Обучения студентов компиляторам в университетах КЗ
- Компиляции C кода в WebAssembly для браузерных приложений
- IDE интеграции через LSP протокол
- Платформы для обучения Rust разработчиков

## Стадия
MVP — продукт работает, компилирует реальный код, имеет GitHub репозиторий с CI.

## Команда
- Руководитель: системный программист, опыт в C/Go/Rust/Asm
- Проекты: NexusEye VMS (66K LOC Go), EntropyX (H.265 транскодер), BinaryAI (CPU LLM inference)
